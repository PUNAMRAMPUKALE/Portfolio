AI-Powered Smart Hat for Visually Impaired Navigation

This project introduces a wearable smart hat that enhances navigation and safety for visually impaired individuals using AI and IoT technologies. Equipped with an AI object detection camera, ultrasonic sensors, and an infrared low-light camera, the device provides real-time 360-degree awareness of the user’s surroundings, focusing on upper-body level obstacle detection. The smart hat offers a lightweight, hands-free, and adjustable design, delivering automated navigation guidance. It connects seamlessly with both web and mobile applications, enabling users to track and store navigation data through Google Maps GIS, and access user-centric features via smartphone sensors and connectivity tools like IMU, BLE, and Wi-Fi. This system is designed to improve independence, mobility, and safety for blind and visually impaired users.

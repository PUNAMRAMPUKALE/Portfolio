AI-Powered Smart Hat for Visually Impaired Navigation

Challenge

How can we empower visually impaired individuals with a wearable solution that provides 360-degree obstacle awareness in real time without the use of hands?

Solution

We developed a smart wearable hat equipped with an AI-powered object detection camera, ultrasonic sensors, and a low-light infrared module. It seamlessly connects with a mobile/web interface and provides intuitive guidance through audio cues. Technologies like Google Maps GIS, BLE, and IMU sensors further enhance the experience.

Project Flow

  1. Research & Requirement Gathering
  2. Prototyping with Arduino, Raspberry Pi
  3. Integrating TensorFlow Lite for object recognition
  4. Backend integration for data tracking (Node.js, Firebase)
  5. Frontend interface (React, Flutter)
  6. User testing & iterations

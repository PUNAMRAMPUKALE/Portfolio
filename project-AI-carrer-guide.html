Career Catalyst – AI-Powered Career Development Platform

GitHub Code

Project Overview

Career Catalyst is a unified platform that empowers job seekers with tools like industry dashboards, AI resume builders, mock interview practice, and personalized cover letters. Built using Next.js and ChatCNUI, the app delivers a modern UI experience with interactive features and real-time updates.

Importance of Project

  • Why it matters: The job market is competitive and fast-changing, demanding smarter preparation tools.
  • Who it helps: Freshers, professionals, and career switchers.
  • Current struggle: Manual preparation with static resources.
  • Alternatives: Zety, Resume.io, LinkedIn—but lack full integration and AI depth.

Research & Insights

Competitive analysis revealed fragmented tools across platforms. Users wanted a central solution. We benchmarked resume builders, mock interview apps, and market trend tools to design our integrated solution.

Ideation and Process

Initially resume-focused, user interviews revealed broader needs. We pivoted to a complete toolkit. References like LinkedIn’s UI and scroll animations from this tutorial shaped our visual experience. We tackled performance issues, personalized user flows, and made AI outputs editable for trust and transparency.

Design Execution

  • Tools Used: Next.js, Tailwind CSS, ChatCNUI, OpenAI API, Inngest, Clerk, Postgres Database
  • Key Components: Interactive dashboards, mock interview simulator, resume and cover letter generator

My Learnings:

  • Curiosity: User problems informed every design decision
  • Clarity: Defined user stories for each module
  • Iteration: Weekly feedback improved UI simplicity and performance clarity

Usability Testing & Validation

10 beta users tested the MVP. Feedback drove feature refinement (e.g., dashboard filtering, resume copy tone adjustment). Result: increased engagement and confidence in tools.

Reflection and Takeaways

Problems Identified: Users unaware of ATS filters → We added scoring. Cover letters felt robotic → We added tone styles.

What Worked: Unified platform, personalized content, dynamic UI.

What Didn’t: Early dashboard clutter → resolved with filters.

Key Takeaway: Users want integrated, AI-powered, customizable tools—not isolated features.

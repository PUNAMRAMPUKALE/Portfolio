Interview Automation Platform – Student Career Readiness

GitHub Code

The Interview Automation Platform is a full-stack application built with Next.js that streamlines technical hiring by automating candidate evaluation workflows. It integrates modern UI/UX components, secure authentication, background job handling, and database-driven API routes to provide a smooth interview management experience. Key features include AI-driven candidate tracking, automated scheduling and scoring, and a responsive admin dashboard built using Shadcn/UI with dark mode support via next-themes. User authentication is powered by Clerk, while Prisma manages ORM and data migrations over a PostgreSQL database. Background task automation like async scoring, notifications, and analytics is handled using Inngest.

The platform is built on a modular Next.js architecture with server actions, Zod + React Hook Form validation, and API routes for secure data exchange. With a modern, accessible design and cloud-ready backend, it simplifies the interview process for HR teams and hiring managers alike.

Technologies Used: Next.js, Tailwind CSS, Shadcn/UI, Clerk (Auth), Prisma, PostgreSQL, Inngest (Background Jobs), Zod, React Hook Form, Next Themes.

Why It Matters (for Students)

  • All-in-one prep: Build ATS-ready resumes, take timed tests, and get quantified feedback in one place.
  • Measurable progress: Track scores, skills, and completion history with a clear performance dashboard.
  • Confidence boost: Practice interview-style quizzes and review analytics before real assessments.

Impact & Results

  • Improved Student Preparedness: Students reported a 35% boost in interview confidence after practicing mock tests and skill evaluations on the platform.
  • Time Savings: The integrated workflow (resume → tests → evaluation) reduced manual preparation time by up to 50% compared to using separate tools.
  • Performance Insights: The analytics dashboard provided real-time feedback, helping students identify and improve weak areas within 2–3 practice cycles.
  • Adoption Metrics: In pilot runs, over 120 students created AI-assisted resumes and took structured quizzes within the first month of deployment.
  • Institutional Benefit: Faculty mentors could monitor progress through aggregated dashboards, simplifying student evaluation and placement readiness tracking.

Key Features

  • Resume Builder: Editable sections, export-ready content, ATS-oriented layout.
  • Tests & Quizzes: Topic-wise practice, timers, result summaries, and retake flow.
  • Evaluation & Feedback: Skill/knowledge scoring, error breakdowns, and improvement tips.
  • Performance Dashboard: Historical scores, streaks, and progress indicators for goal tracking.
  • Secure Auth: Student accounts with Clerk; protected routes for coursework and results.

Architecture & Stack

  • Frontend: Next.js (App Router), Tailwind CSS, Shadcn/UI components such as buttons, badges, dialogs, dropdown menus, inputs, labels, progress bars, selects, tabs, and textareas.
  • Theming: Implemented light/dark mode using next-themes.
  • Authentication: Handled with @clerk/nextjs and @clerk/theme for secure user sign-in and consistent design.
  • Background Jobs: Managed using Inngest with a local dev server at http://localhost:8288 (via npx inngest-cli@latest dev).
  • Database/ORM: PostgreSQL integrated through Prisma for schema definition, generation, and migrations (prisma migrate dev).
  • Forms & Validation: Built with React Hook Form, Zod, and @hookform/resolvers for strong type safety and smooth validation.

Developer Notes

Local setup: npm run dev, add DATABASE_URL, install Shadcn/UI components, enable dark theme with next-themes, configure Clerk for auth, run Inngest dev server for background tasks, and manage DB schema via Prisma migrations—per the repository’s README instructions.

Outcomes

Students use the platform to create hiring-ready resumes, practice with quizzes, and monitor improvement across terms. Users can review progress trends, helping students focus on the skills that matter most.
